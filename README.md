### 💡最大的收获：自学套路

以前大一上学期自学 C 语言和数据结构，习惯跟着 B 站视频走，笔记全记在纸质本上，进度慢不说，想查点东西还特别费劲。 这次训练营我开启了新模式：

- **看文档**：《简明 Python 教程》等
- **问 AI**：不懂的直接丢给AI，再让AI生成笔记，效率起飞
- **md笔记**：从 Day1 系统的整理了 Markdown 语法后，现在已经能熟练地用 Typora、obsidian 把笔记写得漂漂亮亮的

## 🌊 学得越多，越感觉水深

深度学习的“水”很深，学习路线里面的内容都是比较浅显的，而且为了能完成每日任务，很多知识都是囫囵吞枣，一知半解。特别是其中多元微分、线代的数学知识

不过没关系，下学期这些课刚好就要开了。寒假先预习一遍，开学后再好好巩固
