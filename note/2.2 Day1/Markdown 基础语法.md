## 📝 Markdown 基础语法

### 1. 标题 (Headings)

使用 `#` 号加空格，1 到 6 个 `#` 分别对应一到六级标题。

Markdown

```
# 一级标题
## 二级标题
### 三级标题
```

### 2. 文本格式 (Emphasis)

- **加粗**：`**文本**` 或 `__文本__`
- *斜体*：`*文本*` 或 `_文本_`
- ***粗斜体\***：`***文本***`
- ~~删除线~~：`~~文本~~`
- <u>下划线</u>：`<u>文本</u>`

### 3. 列表 (Lists)

#### 无序列表

使用 `*`、`+` 或 `-` 加上空格。

- 项目一
- 项目二

#### 有序列表

数字加上 `.` 再加上空格。

1. 第一步
2. 第二步

### 4. 链接与图片 (Links & Images)

- **链接**：`[显示文本](URL地址)`

  > 例如：[百度](https://www.baidu.com/)     		按下ctrl并点击蓝色链接即可访问

- **图片**：`![替代文本](图片链接地址)`

  > 例如：`![Markdown Logo](https://markdown-here.com/img/icon256.png)`

### 5. 代码 (Code)

- **行内代码**：使用反引号包裹 ``代码``。
- **代码块**：使用三个反引号 ``` 包裹，并可以指定语言进行语法高亮。

Python

```python
def hello_world():
    print("Hello, Markdown!")
```

### 6. 引用 (Blockquotes)

使用 `>` 符号。

> 这是一个引用示例。
>
> > 甚至可以嵌套引用。

### 7. 分割线 (Horizontal Rules)

使用三个或以上的 `***`、`---` 或 `___`。

------

### 8. 表格 (Tables)

使用 `|` 分隔列，使用 `-` 分隔表头和内容。

| **项目** | **价格** | **数量** |
| -------- | -------- | -------- |
| 苹果     | $5       | 10       |
| 香蕉     | $2       | 5        |

> **小技巧**：冒号 `:` 的位置决定了对齐方式（左对齐、居中、右对齐）。

------

### 9.特殊图表的绘制

在 Markdown 中生成流程图、饼图等图表，通常是使用 **Mermaid** 语法[点击跳转](https://github.com/HankLi666/QG/blob/master/note/2.2%20Day1/Mermaid%20%E8%AF%AD%E6%B3%95%E8%A7%A3%E6%9E%90.md)

### 10.小贴士

- **换行**：在一行末尾输入两个空格然后回车，或者直接空出一行。
- **转义**：如果你想在文档里显示这些特殊符号（如 `#` 或 `*`）而不被解析，请在前面加上反斜杠 `\`。
