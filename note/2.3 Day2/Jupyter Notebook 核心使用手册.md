#  Jupyter Notebook 使用手册

##  简介

Jupyter Notebook 是一个基于网页的**交互式计算环境**。它最强大的地方在于可以将 **代码** 、**Markdown**、**数学公式** 和 **可视化图表** 完美融合在同一个文档（`.ipynb` 文件）中。

------

## 核心操作模式

Jupyter 的操作逻辑类似于 Vim 编辑器，分为两种模式。理解颜色变化是上手的关键：

### 1. 命令模式 (Command Mode)

- **外观：** 单元格左侧边框为 **蓝色**。
- **用途：** 用于**单元格级别**的操作（如移动、删除、新增单元格）。
- **进入方法：** 选中单元格后按 **`Esc`**。

### 2. 编辑模式 (Edit Mode)

- **外观：** 单元格左侧边框为 **绿色**，且内部有光标闪烁。
- **用途：** 用于在单元格内**输入内容**（写代码或文档）。
- **进入方法：** 在单元格上按 **`Enter`** 或用鼠标直接点击输入区域。

------

##  必记快捷键指南

熟练使用快捷键可以让你在记录康复计划或写代码时完全脱离鼠标。

### 模式切换与运行

| **快捷键**          | **作用**                           |
| ------------------- | ---------------------------------- |
| **`Esc`**           | 切换到**命令模式** (蓝色)          |
| **`Enter`**         | 切换到**编辑模式** (绿色)          |
| **`Shift + Enter`** | **运行**当前单元格，并跳转到下一格 |
| **`Ctrl + Enter`**  | **运行**当前单元格，运行完停在原地 |

### 单元格操作 (在命令模式下使用)

| **快捷键** | **作用**                               |
| ---------- | -------------------------------------- |
| **`M`**    | 将单元格转换为 **Markdown** (笔记模式) |
| **`Y`**    | 将单元格转换为 **Code** (代码模式)     |
| **`A`**    | 在当前格**上方** (Above) 插入新格      |
| **`B`**    | 在当前格**下方** (Below) 插入新格      |
| **`D, D`** | 连续按两次 D，**删除**当前单元格       |
| **`Z`**    | **撤销**删除操作                       |

***

## 保存与安全关闭

1. **保存：** 按 **`Ctrl + S`**。
2. **正常退出：**
   - 在网页端点击 `File` -> `Close and Halt`（这会停止后台运行的内核）。
   - 在启动 Jupyter 的黑色终端窗口（Anaconda Prompt）中按下 **`Ctrl + C`** 并输入 **`y`**，彻底关闭服务。
