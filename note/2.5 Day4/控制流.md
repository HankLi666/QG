### 1. 条件判断（if / elif / else） while循环

与 C 语言逻辑一致，但语法更精简

**变化点**：`else if` 缩写为 **`elif`**；判断条件**不需要括号**；不使用`{}`改为`:`

```python
#条件判断
score = int(input("分数: "))
if score >= 90:
    print("优秀")
elif score >= 60:    # 相当于 C 的 else if
    print("及格")
else:
    print("不及格")
  
#while循环
count = 5 
while count > 0: 
	print(count) 
	count -= 1 
```

### 2. 循环结构：for ... in

这是 C 与 Python 差异最大的地方。C 的 `for` 本质是**计数控制**，而 Python 的 `for` 本质是**序列遍历**（Iterator）。

#### 1. 遍历容器（直接取值）

在 C 语言中，你需要通过下标 `i` 访问数组；在 Python 中，`for` 直接把容器里的“货物”拿出来。

```python
names = ["Alice", "Bob", "Charlie"]
for name in names:
    print(name)  # 直接打印出名字，不需要 names[i]
```
#### 2. 配合 `range()` 模拟 C 的索引循环

如果需要像 C 语言 `for(int i=0; i<10; i++)` 那样计数，Python 使用 `range()` 构造一个序列。

- `range(stop)`：生成 $0$ 到 $stop-1$ 的stop个整数。
    
- `range(start, stop, step)`：指定起始、终点和步长。

```python
# 相当于 for(int i=0; i<5; i++)
for i in range(5):
    print(i) # 输出 0, 1, 2, 3, 4

# 相当于 for(int i=10; i>0; i-=2)
for i in range(10, 0, -2):
    print(i) # 输出 10, 8, 6, 4, 2
```

#### 3.成员运算符 `in` 的逻辑

`in` 关键字不仅用于 `for` 循环，还可以单独用于判断，返回布尔值（True/False）。

```python
fruits = ["apple", "banana"]
if "apple" in fruits:
    print("有苹果！")
```
#### 4.  进阶：列表推导式 (List Comprehension)

本质是**一种从旧列表快速生成新列表的简写方式**
- **C 语言逻辑**：声明新数组 -> 写 `for` 循环 -> 满足条件则 `append`。
- **Python 推导式**：`[新元素表达式 for 变量 in 可迭代对象 if 条件]`
```python
# 语法：[表达式 for 变量 in 容器 if 条件]
squares = [i**2 for i in range(5)]  # [0, 1, 4, 9, 16]

# 还可以带判断：只保留偶数的平方
even_squares = [i**2 for i in range(10) if i % 2 == 0]
```

### 3. 循环控制：break / continue / pass

- `break` 和 `continue`：用法与 C 语言完全相同。
    
- **`pass` (新增)**：空语句占位符。因为 Python 没有大括号，如果你写了一个 `if` 但还没想好内容，必须写个 `pass`，否则会报语法错误。

```python
if x > 0:
	pass  # 以后再写逻辑，现在先别报错
```
