### 1. 基础定义与 DocStrings（文档字符串）

Python 使用 `def` 关键字。**DocStrings** 是紧跟在函数定义后的字符串，用于解释函数功能，这在 C 语言中通常只能用注释实现。

```python
def greet(name):
    """
    向指定的人发送问候。
    参数 name: 接收者的姓名
    """
    print(f"Hello, {name}!")

# 可以通过 help(greet) 或 greet.__doc__ 查看文档说明
```

### 2. 参数的多种玩法 

#### 1. 默认参数值（Default Arguments）

允许在定义时给参数一个默认值。调用时如果不传该参数，则使用默认值。

> **注意**：默认参数必须放在非默认参数的**后面**。

```python
def power(x, n=2):
    return x ** n

print(power(5))    # 输出 25 (使用默认值 n=2)
print(power(5, 3)) # 输出 125 (覆盖默认值)
```

#### 2. 关键字参数（Keyword Arguments）

调用函数时，可以通过 `参数名=值` 的形式传参，这样**参数顺序可以乱序**。

```python
def describe_pet(animal, name):
    print(f"{name} 是一只 {animal}")

describe_pet(name="大黄", animal="小狗") # 顺序不重要，可读性极高
```

#### 3. 可变参数（Arbitrary Arguments）

如果不确定会传入多少个参数，可以使用 `*` 或 `**`。

- **`*args`**：接收任意多个位置参数，打包成一个**元组 (tuple)**。
```python
def make_pizza(size, *toppings):
    print(f"制作一个 {size} 英寸的披萨，配料包括: {toppings}")

make_pizza(12, '香肠', '蘑菇', '芝士') # '香肠'等会被打包进 toppings
```

- __`**kwargs`__：接收任意多个关键字参数，打包成一个**字典 (dict)**。
```python
def build_profile(first, last, **user_info):
    # user_info 会变成 {'location': '北京', 'field': '计算机'}
    print(f"用户: {first} {last}")
    print(f"额外信息: {user_info}")

build_profile('张', '三', location='北京', field='计算机') #打包进user_info
```

### 3. 变量作用域与 `global` 关键字

在 Python 中，函数内部可以直接读取全局变量，但如果你想在函数内**修改**全局变量，必须使用 `global` 声明。

- **C 语言对比**：C 语言里全局变量在任何地方都能改，Python 增加 `global` 是为了防止你在函数内不小心误改了全局数据。

```python
count = 10

def update_count():
    global count  # 必须声明，否则 Python 会把它当成一个新的局部变量
    count = 20
    print(f"函数内修改为: {count}")

update_count()
print(f"全局变量现在是: {count}")
```

### 4. 函数式编程工具：Lambda 与高阶函数

在 Python 中，函数可以像变量一样被传递。这使得可以写出极其简洁的代码。
#### 1. 匿名函数 `lambda`

`lambda` 允许你定义一个**没有名字、只有一行代码**的微型函数。它就像 C 语言中为了一个简单的回调逻辑而临时写的“内联逻辑”。

- **语法**：`lambda 参数: 表达式` (自动返回结果)
```python
# 普通函数
def add(x, y): return x + y

# lambda 形式
add_lambda = lambda x, y: x + y
print(add_lambda(3, 5)) # 输出 8
```

#### 2. 高阶函数：`map()` 与 `filter()`

“高阶函数”是指**接收另一个函数作为参数**的函数。它们是 Python 处理序列（列表等）的利器。

- **`map(function, iterable)`**：将函数作用于序列的**每一个**元素，返回加工后的结果。

> **类比**：给工厂流水线上的每一个零件都打上标签。

- **`filter(function, iterable)`**：根据函数的返回值（True/False）**过滤**掉不符合条件的元素。

> **类比**：通过一个筛子，只保留符合规格的零件。

```python
nums = [1, 2, 3, 4, 5, 6]

# 使用 map 批量计算平方
squares = list(map(lambda x: x**2, nums)) 
# 结果: [1, 4, 9, 16, 25, 36]

# 使用 filter 筛选偶数
evens = list(filter(lambda x: x % 2 == 0, nums)) 
# 结果: [2, 4, 6]
```
