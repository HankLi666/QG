## 数据处理
### 数据合并与创建矩阵
- **合并数据 (`np.hstack`)**： 将分布在不同行或列的特征（X）横向拼接成一个完整的矩阵。
```python
X = np.hstack([data.values[::2, :], data.values[1::2, :2]])
y = data.values[1::2, 2]
```
- **拼接列名**：
```python
data = pd.DataFrame(complete_data, columns=columns)
```
### 提取数据
`.values` 将 Pandas 表格转回 **Numpy 数组**，才能进行矩阵运算
- `X = data.drop('TARGET', axis=1).values`：剥离目标值，提取特征矩阵
- `y = data['TARGET'].values`：提取标签向量

## 实验设置
### 设置随机种子 
`np.random.seed`  固定随机数生成的顺序
- **意义**：保证每次运行代码时，洗牌的结果、权重的初始化都是一样的。这样当修改模型参数时，能确定准确率的提升是来自优化，而不是运气
### 数据洗牌
防止原始数据的排序规律（如按房价从高到低）误导模型
```python
indices = np.arange(X.shape[0])
np.random.shuffle(indices)
```
### 数据切分
常按 **8:2** 比例划分为**训练集**和**测试集**
```python
split_idx = int(X.shape[0] * 0.8)
train_idx, test_idx = indices[:split_idx], indices[split_idx:]
```

## 线性回归类封装
sklearn接口形式：

| **函数**     | **功能**                                        | **对应概念**   |
| ---------- | --------------------------------------------- | ---------- |
| `__init__` | **初始化**。设置学习率 `lr`、迭代次数 `epochs` 等超参数         | 定义网络结构与超参数 |
| `fit`      | **训练**。根据输入 $X$ 和 $y$，利用解析解或梯度下降求出最优权重 $w$    | 反向传播与优化    |
| `predict`  | **预测**。利用学到的 $w$ 对新数据 $X$ 进行计算：$\hat{y} = Xw$ | 前向传播       |
| `score`    | **评估**。计算 $R^2$ 或 MSE，衡量模型在未知数据上的表现           | 损失函数评估     |
