## 1.Matplotlib 库的导入

```python
import matplotlib.pyplot as plt
```
- **`.pyplot`** 是其中最常用的绘图模块，它包含了大部分类似“画线”、“加标题”的简单命令

---
## 2.核心绘图逻辑：层级结构

Matplotlib 的设计像是一个“套娃”结构：

- **Figure (画布)**：最顶层。你可以把它看作一张白纸。

- **Axes (子图/坐标系)**：纸上的绘图区域。一张画布可以包含多个坐标系（比如左边画 Loss，右边画 Accuracy）。

- **Axis (坐标轴)**：具体的 X 轴和 Y 轴。

- **Artist (图形元素)**：你在坐标系里画的线、点、文字、图例等。

### `plt.figure()`：创建画布

这是绘图的第一步，手动创建可以控制图片的大小、分辨率和背景。

#### 常用参数：

- **`figsize`**: 一个元组，控制画布的宽和高

- **`dpi`**: 分辨率

- **`facecolor`**: 画布的背景颜色（用 RGB 或颜色名）

```python
# 创建一个宽 8 英寸、高 4 英寸，背景为浅灰色的画布
plt.figure(figsize=(8, 4), dpi=100, facecolor='#f0f0f0')
```

## 3..常用函数
### 图表生成

| **函数**          | **图表名称**   |
| --------------- | ---------- |
| `plt.plot()`    | **折线图**    |
| `plt.scatter()` | **散点图**    |
| `plt.bar()`     | **条形图**    |
| `plt.hist()`    | **直方图**    |
| `plt.pie()`     | **扇形图/饼图** |
以为例 `plt.plot()`，其他函数仅有细微差别
#### 基本语法：

- `plt.plot(x, y, format_string, **kwargs)`

#### 常用参数：

- **`x`, `y`**: 数据源（通常是 NumPy 数组或列表）。

- **`label`**: 给这条线起个名字（用于显示图例 `legend`）。

- **`color`**: 线条颜色

- **`linewidth` (或 `lw`)**: 线条宽度。

- **`linestyle` (或 `ls`)**: 线条样式。实线 `'-'`, 虚线 `'--'`, 点线 `':'`。

- **`marker`**: 数据点的标记形状。如圆圈 `'o'`, 星号 `'*'`。
#### 其他绘图函数：

| **函数**              | **语法签名 (核心部分)** | **必传参数的含义**                       |
| ------------------- | --------------- | --------------------------------- |
| **`plt.scatter()`** | `(x, y)`        | 与 `plot` 一致，但渲染为独立的点。             |
| **`plt.bar()`**     | `(x, height)`   | `x` 是柱子的位置（或名称），`height` 是柱子的高度。  |
| **`plt.hist()`**    | **`(x)`**       | **只需传一个数组**！它会自动统计 `x` 中数值出现的频率。  |
| **`plt.pie()`**     | **`(x)`**       | **只需传一个数组**！它会计算 `x` 中每个值占总和的百分比。 |
### 图表装饰 

- `plt.title()`: 设置图表总标题。

- `plt.xlabel()` / `plt.ylabel()`: 设置横纵坐标轴的标签。

- `plt.legend()`: 显示图例（当一张图有多条线时非常重要）。

- `plt.grid()`: 开启网格（便于观察数据的具体数值）。

- `plt.xlim()` / `plt.ylim()`: 限制坐标轴的显示范围。
