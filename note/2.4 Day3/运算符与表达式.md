### 1. 算术运算符

Python 的算术运算比 C 语言更直观，尤其是处理除法时。

|**运算符**|**描述**|**示例**|**C 语言差异**|
|---|---|---|---|
|`+`|加|`1 + 1 = 2`|一致|
|`-`|减|`5 - 2 = 3`|一致|
|`*`|乘|`3 * 4 = 12`|一致|
|`/`|**浮点除**|`5 / 2 = 2.5`|**注意！** Python 中即使是两个整数相除，结果也是浮点数|
|`//`|**整除（地板除）**|`5 // 2 = 2`|类似于 C 语言的整数除法，舍弃小数部分|
|`%`|取模（取余）|`5 % 2 = 1`|一致|
|`**`|**幂运算**|`2 ** 3 = 8`|C 语言需要调用 `pow()` 函数，Python 直接内置|

---

### 2. 比较运算符

结果为布尔值：`True` 或 `False`（注意首字母大写，C 语言通常用 0 和 1）。

- `==` (等于), `!=` (不等于)

- `>` , `<` , `>=` , `<=`

- **Python 特色：支持链式比较**
```python
x = 15
print(10 < x < 20)  # 结果为 True，这在 C 语言中是不合法的（会得到错误逻辑结果）
```
---
### 3. 逻辑运算符（语义化）

Python 抛弃了 C 语言的 `&&`, `||`, `!`，改用更接近英语的单词。

|**运算符**|**描述**|**C 语言对应**|**示例**|
|---|---|---|---|
|**`and`**|逻辑与|`&&`|`True and False` -> `False`|
|**`or`**|逻辑或|`||
|**`not`**|逻辑非|`!`|`not True` -> `False`|

---

### 4. 赋值运算符

除了基础的 `=`，Python 也支持复合赋值：

- `+=`, `-=`, `*=`, `/=`, `//=`, `**=`, `%=`

- **注意**：Python 中 **没有** `i++` 或 `i--`！请使用 `i += 1`。


---

### 5. 成员运算符（Python 独有）

这是 Python 极其强大的地方，用于检查一个值是否存在于序列（字符串、列表等）中。

```python
name = "Hank"
print("G" in name)    # Faulse
print("a" not in name) # True
```

---
### 6. 优先级简表

从高到低排列（常用）：

1. **括号** `()`：永远优先级最高。

2. **幂运算** `**`

3. **乘除取整取余** `*`, `/`, `//`, `%`

4. **加减** `+`, `-`

5. **比较运算符** `<`, `<=`, `>`, `>=`, `==`, `!=`

6. **逻辑运算符** `not` > `and` > `or`

### 7.`*` 号的魔法
在 Python 中，`*` 运算符不仅能做数学乘法，还能进行 **“序列重复”**。

#### 1. 数值乘法

当操作数都是数字时，执行标准数学运算。

```python
print(2 * 3)  # 输出: 6
```

#### 2. 字符串/序列重复

当一个操作数是 **字符串**，另一个是 **整数** 时，`*` 会将字符串重复指定次数并拼接起来。


```python
# 基础用法
print('la' * 3)       # 输出: 'lalala'

# 结合 print 函数
print("-" * 20)      # 输出: -------------------- (快速画分割线)

# 顺序无关紧要
print(3 * "abc")     # 输出: 'abcabcabc'
```

> **⚠️ 注意**：乘数必须是**整数**（int）。如果你尝试 `'la' * 3.0`，Python 会报错 `TypeError`，因为它不知道如何将字符串重复“3.0次”。
---
### 💡 避坑指南（C 语言转 Python）

1. **除法的陷阱**：

   在 C 语言中 `5 / 2` 得到 `2`；在 Python 中 `5 / 2` 得到 `2.5`。如果你想要整数结果，必须用 `5 // 2`。

2. **布尔值大小写**：

    Python 的布尔值必须是 `True` 和 `False`，不能写成小写的 `true/false`。

3. **自增自减**：

    看到 `i++` 编译器会报错。记住，Python 提倡“显式优于隐式”，写 `i += 1` 即可。
