### 1. 字符串 (String) 

与 C 语言字符数组最大的不同：Python 字符串是**不可变**的（immutable）

- **定义方式**：使用单引号 `'`、双引号 `"` 或三引号 `'''/"""`（支持多行）。
- **特性**：支持切片，拥有丰富的内置方法。

- **常用操作**：
```python
s = " Python Programming "
print(s.strip())       # 去除两端空格
print(s.lower())       # 全小写
print(s.split(" "))    # 按空格分割，返回列表
print(s[::-1])         # 字符串反转（切片魔法）
```

### 2. 列表 (List) - 强大的动态数组

对应 C 语言的数组，但它可以存储**不同类型**的对象，且长度**自动伸缩**。
-  **定义方式**：使用方括号 `[]`，元素间用逗号分隔。
- **特性**：有序、可变（mutable）、允许重复。
- **常用操作**：
```python
nums = [1, 2, 3]       # 定义列表
nums.append(4)         # 末尾添加: [1, 2, 3, 4]
nums.insert(0, 0)      # 指定位置插入: [0, 1, 2, 3, 4]
nums[1] = "A"          # 修改元素
last = nums.pop()      # 弹出末尾元素
```

### 3. 元组 (Tuple) - 锁定的列表

元组就像是一个“只读”的列表。

- **特性**：有序、**不可变**（一旦创建不可修改）。

- **语义**：通常用于存储结构固定的数据（如坐标 `(x, y)`，数据库记录）。

- **为什么用它？** 比列表更快，且可以作为字典的“键”。

```python
point = (10, 20)
# point[0] = 100  # 报错！不可修改

 single_tuple = (5,)
 # 特别注意：定义只有一个元素的元组，必须加逗号
```

### 4. 字典 (Dict) - 键值对哈希表

对应 C 语言中需要复杂实现的 Hash Table。
- **定义方式**：使用花括号 `{}`，内部为 `key: value` 形式。
- **特性**：通过 `Key` 查找 `Value`，极快（$O(1)$复杂度）；Key 必须是不可变类型。
- **常用操作**：

```python
user = {"name": "Hank", "age": 18}
print(user["name"])    # 获取值
user["score"] = 100    # 新增或修改
del user["age"]        # 删除键值对
print(user.keys())     # 获取所有键
```

### 5. 集合 (Set) - 无序去重容器

类似于数学中的集合概念。
- **定义方式**：使用花括号 `{}` 或 `set()`。
- **特性**：**元素唯一**、无序。

- **常用操作**：用于去重或交、并、差集运算。
```python
empty_set = set()      # 注意：定义空集合必须用 set()，因为 {} 是留给空字典的 
s1 = {1, 2, 2, 3}      # 自动去重，结果为 {1, 2, 3}
s2 = {3, 4, 5}
print(s1 & s2)         # 交集: {3}
print(s1 | s2)         # 并集: {1, 2, 3, 4, 5}
print(s1 - s2)         # 差集: {1，2，4，5}
```

### 数据类型对比表

| **数据结构**         | **符号**  | **可变性** | **有序性** | **访问方式**  | **对应 C 语言概念**       | **核心用途与特性**         |
| ---------------- | ------- | ------- | ------- | --------- | ------------------- | ------------------- |
| **字符串 (String)** | `""`    | **不可变** | 是       | 索引 `s[i]` | `char[]` / `string` | 文本处理、信息提取与清洗        |
| **列表 (List)**    | `[]`    | **可变**  | 是       | 索引 `l[i]` | 动态数组 / 链表           | 存储有序清单，支持动态增删       |
| **元组 (Tuple)**   | `()`    | **不可变** | 是       | 索引 `t[i]` | `const array`       | 锁定坐标、常量配置，安全性高      |
| **字典 (Dict)**    | `{k:v}` | **可变**  | 3.7+有序  | 键 `d[k]`  | 哈希表 (Hash Table)    | 结构化档案管理，$O(1)$ 极速查找 |
| **集合 (Set)**     | `{}`    | **可变**  | 否       | 遍历 / 运算   | 唯一索引表               | 自动去重、集合运算（交并差）      |
