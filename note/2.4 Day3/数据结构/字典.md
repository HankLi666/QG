
字典以「键值对（key-value pairs）」为核心，支持高效的创建、查询、更新、删除等操作
其中
1. 键（key）必须**唯一**且**可哈希**（如字符串、数字、元组，不能用列表、字典等可变类型）；
2. 值（value）可以是任意 Python 类型（包括列表、字典、函数等）；
### 一、基础功能：创建字典

| 功能           | 语法 / 函数                           | 示例代码                                                                                       |
| :----------- | :-------------------------------- | :----------------------------------------------------------------------------------------- |
| 直接创建（键值对）    | `dict = {key1: val1, key2: val2}` | `my_dict = {"name": "Bob", "age": 28, "city": "Beijing"}`                                  |
| 创建空字典        | `dict = {}` 或 `dict = dict()`     | `empty_dict = {}` 或 `empty_dict = dict()`                                                  |
| 用列表 / 元组批量创建 | `dict(可迭代对象)`                     | `dict_from_list = dict([("a", 1), ("b", 2)])` # 结果：{"a":1, "b":2}                          |
| 带默认值的字典      | `collections.defaultdict(类型)`     | `from collections import defaultdict`  <br>`default_dict = defaultdict(int)` # 键不存在时默认值为 0 |
| 键相同的字典       | `dict.fromkeys(键列表, 默认值)`         | `dict_from_keys = dict.fromkeys(["x", "y"], 0)` # 结果：{"x":0, "y":0}                        |

### 二、核心功能：访问 / 查询值

通过「键」获取对应的值，支持安全查询（避免键不存在报错）：

|功能|语法 / 函数|示例代码|
|:--|:--|:--|
|直接通过键访问|`dict[key]`|`print(my_dict["name"])` # 输出：Bob（键存在时返回值）|
|安全访问（推荐）|`dict.get(key, 默认值)`|`print(my_dict.get("gender", "未知"))` # 输出：未知（键不存在返回默认值）|
|检查键是否存在|`key in dict` / `key not in dict`|`print("age" in my_dict)` # 输出：True（返回布尔值）|
|获取所有键|`dict.keys()`|`print(my_dict.keys())` # 输出：dict_keys (["name", "age", "city"])`|
|获取所有值|`dict.values()`|`print(my_dict.values())` # 输出：dict_values (["Bob", 28, "Beijing"])`|
|获取所有键值对|`dict.items()`|`print(my_dict.items())` # 输出：dict_items ([("name","Bob"), ("age",28)])`|

### 三、核心功能：添加 / 修改键值对

字典是「可变类型」，支持动态添加新键值对、修改已有键的值：

|功能|语法 / 函数|示例代码|
|:--|:--|:--|
|修改已有键的值|`dict[key] = 新值`|`my_dict["age"] = 29` # 原键 "age" 的值从 28 改为 29，结果：{"name":"Bob", "age":29, ...}`|
|添加新键值对|`dict[新key] = 新val`|`my_dict["gender"] = "Male"` # 新增键 "gender"，结果：{... "gender":"Male"}`|
|批量添加 / 修改|`dict.update(其他字典/可迭代对象)`|`my_dict.update({"city": "Shanghai", "hobby": "跑步"})` # 修改 "city"，新增 "hobby"`|

### 四、核心功能：删除元素

支持删除单个键值对、清空字典、删除整个字典：

|功能|语法 / 函数|示例代码|
|:--|:--|:--|
|删除指定键值对（返回值）|`dict.pop(key, 默认值)`|`age_val = my_dict.pop("age")` # 删除 "age" 键，返回其值 29；键不存在时返回默认值（可选）|
|删除最后一个键值对（Python 3.7+）|`dict.popitem()`|`last_item = my_dict.popitem()` # 输出：("hobby", "跑步")（删除并返回最后一个键值对）|
|清空所有键值对|`dict.clear()`|`my_dict.clear()` # 结果：{}（空字典，字典对象仍存在）|
|删除整个字典|`del dict`|`del my_dict` # 彻底删除字典对象，后续无法再访问 my_dict|

### 五、进阶功能：其他常用操作

| 功能            | 语法 / 函数                      | 示例代码                                                                             |
| :------------ | :--------------------------- | :------------------------------------------------------------------------------- |
| 复制字典（浅拷贝）     | `dict.copy()` 或 `dict(dict)` | `new_dict = my_dict.copy()` # 新字典修改不影响原字典（浅拷贝：仅拷贝表层键值对）                          |
| 深拷贝（含嵌套字典）    | `copy.deepcopy(dict)`        | `from copy import deepcopy`  <br>`deep_new_dict = deepcopy(my_dict)` # 嵌套结构也完全拷贝 |
| 获取字典长度（键值对个数） | `len(dict)`                  | `print(len(my_dict))` # 输出：3（my_dict 有 3 个键值对时）                                  |
| 字典推导式（快速构建字典） | `{k: v for k, v in 可迭代对象}`   | `square_dict = {x: x*x for x in range(1, 5)}` # 结果：{1:1, 2:4, 3:9                |
