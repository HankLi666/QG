#### 元组功能特性（核心：不可变）

- 不可修改、添加、删除单个元素（仅支持`count()`/`index()`等查询类方法）；
- 可哈希（当元素均为可哈希类型时，可作为字典的键）；
- 内存占用较小（不可变，存储更紧凑）；
- 适用场景：存储**固定数据**（不允许修改的内容，如坐标、配置参数）

#### 使用赋值运算符直接创建元组
创建元组时，也可以使用赋值运算符 “=” 直接将一个元组赋值给变量。语法格式如下：
```python
tuplename = (element 1,element 2,element 3,……,element n)
```
元组元素的个数没有限制，但不能被修改
#### 创建数值元组
可以使用 tuple () 函数直接将 range () 函数循环出来的结果转换为数值元组。tuple () 函数的基本语法如下：
```python
tuple (data)
```
####  删除元组
对于已经创建的元组，不再使用时，可以使用 del 语句将其删除。语法格式如下：
 ```python
 del tuplename
 ```
 其中，tuplename 为要删除元组的名称
