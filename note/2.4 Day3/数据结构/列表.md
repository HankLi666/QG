#### 列表功能特性（核心：可变）

- 可直接修改、添加、删除元素（支持`append()`/`remove()`/`sort()`等丰富方法）；
- 内存占用较大（需预留动态扩容空间）；
- 不可哈希（不能作为字典的键）；
- 适用场景：存储**动态数据**（需增删改的内容，如用户输入、待办事项）

####  创建列表
同其他类型的 Python 变量一样，创建列表时，也可以使用赋值运算符 “=” 直接将一个列表赋值给变量，语法格式如下：  
```python
listname = [element 1，element 2，element 3，…，element n]
```
#### 创建空列表  
在 Python 中，也可以创建空列表，例如，要创建一个名称为 emptylist 的空列表，可以使用下面的代码： 
```python
emptylist = []
```
以下是列表基础操作

| 操作类型     | 具体操作          | 语法 / 函数                                         | 说明                                             | 示例代码（以 `my_list = [1,2,3]` 为例）             | 输出结果                |
| -------- | ------------- | ----------------------------------------------- | ---------------------------------------------- | ------------------------------------------ | ------------------- |
| 访问 / 查询  | 按索引取元素        | `my_list[索引]` / `my_list[反向索引]`                 | 正向索引从`0`开始，反向索引从`-1`开始（对应最后一个元素）               | `my_list[0]` / `my_list[-1]`               | `1` / `3`           |
| 访问 / 查询  | 判断元素是否存在      | `元素 in/not in my_list`                          | 返回布尔值（`True`/`False`），判断元素是否包含在列表中             | `2 in my_list` / `4 not in my_list`        | `True` / `True`     |
| 增加元素     | 末尾添加 1 个元素    | `my_list.append(元素)`                            | 直接在列表末尾添加，元素可为任意类型（字符串、数字、列表等）                 | `my_list.append(4)` → `print(my_list)`     | `[1,2,3,4]`         |
| 增加元素     | 末尾添加多个元素      | `my_list.extend(可迭代对象)`                         | 接收列表、字符串等，将其拆分为单个元素添加（区别于`append`）             | `my_list.extend([4,5])` → `print(my_list)` | `[1,2,3,4,5]`       |
| 增加元素     | 指定位置插入元素      | `my_list.insert(索引, 元素)`                        | 在指定索引处插入，原元素自动后移（索引超界则插入末尾）                    | `my_list.insert(1, 0)` → `print(my_list)`  | `[1,0,2,3]`         |
| 修改元素     | 按索引修改元素       | `my_list[索引] = 新值`                              | 直接覆盖指定索引位置的元素                                  | `my_list[2] = 9` → `print(my_list)`        | `[1,2,9]`           |
| 删除元素     | 按索引删除（无返回）    | `del my_list[索引]`                               | 彻底删除指定索引的元素，不保留被删内容                            | `del my_list[1]` → `print(my_list)`        | `[1,3]`             |
| 删除元素     | 按值删除（删第一个匹配项） | `my_list.remove(元素)`                            | 找到第一个匹配的元素删除，元素不存在会报错                          | `my_list.remove(2)` → `print(my_list)`     | `[1,3]`             |
| 删除元素     | 按索引删除（返回元素）   | `my_list.pop(索引)`                               | 默认删除最后一个元素，返回被删除的内容（可指定索引）                     | `my_list.pop()` → `print(结果, my_list)`     | `3` / `[1,2]`       |
| 删除元素     | 清空列表          | `my_list.clear()`                               | 保留列表对象，删除所有元素（变为空列表）                           | `my_list.clear()` → `print(my_list)`       | `[]`                |
| 切片（批量操作） | 批量获取元素        | `my_list[start:end:step]`                       | `start`默认`0`，`end`默认列表长度（不包含），`step`为步长（默认`1`） | `my_list[0:2]` / `my_list[::-1]`           | `[1,2]` / `[3,2,1]` |
| 统计 / 判断  | 获取列表长度        | `len(my_list)`                                  | 返回列表中元素的总个数                                    | `len(my_list)`                             | `3`                 |
| 统计 / 判断  | 统计元素出现次数      | `my_list.count(元素)`                             | 计算指定元素在列表中的出现次数（无则返回`0`）                       | `[1,2,2,3].count(2)`                       | `2`                 |
| 排序 / 反转  | 原地排序（升序 / 降序） | `my_list.sort()` / `my_list.sort(reverse=True)` | 直接修改原列表，仅支持同类型元素排序（如全数字、全字符串）                  | `num_list=[3,1,2]` → `num_list.sort()`     | `[1,2,3]`           |
| 排序 / 反转  | 反转列表顺序        | `my_list.reverse()`                             | 直接修改原列表，将元素顺序颠倒                                | `my_list.reverse()` → `print(my_list)`     | `[3,2,1]`           |
| 拼接 / 重复  | 拼接两个列表        | 列表1 + 列表2                                       | 返回新列表，原列表保持不变                                  | `[1,2] + [3,4]`                            | `[1,2,3,4]`         |
| 拼接 / 重复  | 重复列表元素        | 列表 * n（n 为整数)                                   | 将列表元素重复`n`次，返回新列表                              | \[1,2] * 2                                 | \[1,2,1,2]          |
