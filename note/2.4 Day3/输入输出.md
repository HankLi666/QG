### input 函数（接收输入）

- 作用：获取用户键盘输入，**默认返回字符串类型**。
- 注意：需强制转换类型（如`int`/`float`）才能赋值给非字符串变量。
- 示例：
```python
# 获取年龄（转换为整数） 
age = int(input("请输入年龄：")) 
# 获取身高（转换为浮点数） 
height = float(input("请输入你的身高（米）：")) 
# 验证类型 
print(f"你的年龄是 {age}，明年你将 {age + 1} 岁。")
```

## print 函数（输出内容）

### 1. 基础输出

与 C 语言不同，Python 的 `print()` 默认会在输出结束后**自动换行**，且不需要指定数据类型。

```python
print("Hello, Python!")  # 输出字符串
print(100)               # 输出数字
print([1, 2, 3])         # 输出列表（C 语言中这需要循环遍历）
```

---
### 2. 多参数输出（自动空格）

可以一次性传入多个对象，`print()` 默认会用**空格**将它们连接起来。

```python
name = "Hank"
age = 1
print("Name:", name, "Age:", age) 
# 输出: Name: Hank Age: 1
```

---
### 3. 三引号实现换行输出

这是 Python 的一大特色。使用 `'''` 或 `"""` 包裹文本，可以保持原有的排版格式（换行、空格等），无需手动打 `\n`。
#### 1. 直接输出多行文本

```python
print('''这是第一行内容
这是第二行内容
这是自动换行的第三行''')
```

#### 2. 结合 f-string 的多行输出

这种方式在生成报告、提示信息或 HTML 模板时极其高效。

```python
name = "小明"
age = 13
height = 1.75

print(f'''用户信息：
姓名：{name}
年龄：{age}
身高：{height}米''')
```
---

### 4. 自定义分隔符 `sep`

如果你不想用空格分隔，可以使用 `sep` 参数（类似于 C 语言里在 `%d` 之间加符号）。

```python
# 用连接符将日期连起来
print("2026", "02", "04", sep="-")
# 输出: 2026-02-04
```

---
### 5. 自定义结束符 `end`

如果你不想换行，或者想以特定字符结尾，修改 `end` 参数（默认是 `\n`）。


```python
print("Hello", end=" ")
print("World")
# 输出: Hello World (在同一行)
```

---
### 6. 使用 `+` 号拼接（类型必须统一）

在 Python 中，`+` 号用于字符串拼接时，要求两边**必须都是字符串**。

> **注意**：这与 C 语言不同，Python 不会自动把数字隐式转换为字符串。

- **错误示范**：`print("年龄是" + 18)` 会报错 `TypeError`。
    
- **正确做法**：使用 `str()` 函数转换。
```python
age = 13
# 注意：Python 的强制转换格式是 str(变量)，而不是 (str)变量
print("年龄是" + str(age))
```
---
### 7. 字符串格式化

**f-string** (Formatted String) 是现代 Python 的标准写法。通过在引号前加 `f`，可以在 `{}` 中直接嵌入变量或表达式。

|**方法**|**示例**|**说明**|
|---|---|---|
|**f-string** (推荐)|`print(f"I am {age} years old")`|Python 3.6+ 加入，最简洁高效|
|**format()**|`print("Hello, {}".format(name))`|比较灵活，适合长文本|
|**% 占位符**|`print("Score: %d" % score)`|**最接近 C 语言**的方式，但不推荐常用|

---
### 8. 进阶：格式化数字（精度控制）

就像 C 语言里的 `%.2f`，在 Python 的 f-string 中也非常简单。

```python
pi = 3.1415926
print(f"Pi is approximately {pi:.2f}") # 保留两位小数
# 输出: Pi is approximately 3.14
```

---

### 9. 写入文件

`print()` 甚至可以直接把内容写进文件，而不需要专门调用 `write()`。

```python
with open("test.txt", "w") as f:
    print("这条信息会被写入文件而非控制台", file=f)
```
