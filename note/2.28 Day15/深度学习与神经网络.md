## 深度学习
- **定义**：机器学习的一个特殊子集，专门使用**多层人工神经网络**来处理复杂模式。

### 基础架构

1. **输入层**：接收原始数据（如像素、特征数值）。
	
2. **隐藏层**：神经网络的“黑盒”，负责自动提取特征。层数越多，提取的特征越抽象。
	
3. **输出层**：给出最终预测结果
---
## 向前传播（推理）
这是模型**做预测**的过程。数据从输入层开始，一层一层向后传递，每一层都对数据进行数学变换（加权求和 + 激活），直到在输出层产生一个预测值 $\hat{y}$
 >使用**矩阵乘法**，高效运行神经网络

### 神经元
神经网络的最小单元。它模拟生物神经元的工作：
- 接收多个输入信号 $x_1, x_2, \dots$
- 每个输入都有一个权重 $w$（表示重要程度）和一个偏置 $b$的参数计算得出输出
#### 工作步骤

- **线性变换**：计算 $z = w^Tx + b$。

- **非线性映射**：将 $z$ 投入激活函数 $a = g(z)$。
### 激活函数
 
[为什么需要激活函数_哔哩哔哩_bilibili](https://www.bilibili.com/video/BV1owrpYKEtP?vd_source=36536e872e00e686f139238779d5d198&spm_id_from=333.788.player.switch&p=59)
激活函数为网络引入了**非线性**，使其能够拟合现实世界中复杂的曲线和模式。如果没有激活函数，无论神经网络有多少层，它本质上都只是一个复杂的“线性模型”。

**主流激活函数对比：**

| **函数**         | **数学表达式**                               | **使用建议**                         |
| -------------- | --------------------------------------- | -------------------------------- |
| **Sigmoid**    | $a = \frac{1}{1+e^{-z}}$                | 二分类问题预测概率                        |
| **ReLU**       | $a = \max(0, z)$                        | **默认首选**。计算极快，能有效缓解深层网络中的梯度消失问题。 |
| **Leaky ReLU** | $a = \max(0.01z, z)$                    | 解决了 ReLU 在 $z<0$ 时神经元“死亡”的问题。    |
| **Tanh**       | $a = \frac{e^z - e^{-z}}{e^z + e^{-z}}$ | 效果通常优于 Sigmoid，因为它将均值中心化到了 0。    |
### 网络层的传播

当神经元组成网络时，使用**向量化**来批量计算。

- **第 1 层**：$Z^{[1]} = W^{[1]}X + b^{[1]} \rightarrow A^{[1]} = g(Z^{[1]})$

- **第 2 层**：$Z^{[2]} = W^{[2]}A^{[1]} + b^{[2]} \rightarrow A^{[2]} = g(Z^{[2]})$

- **输出层**：以此类推，上一层的输出 $A$ 是下一层的输入

## 反向传播(改错)
模型**自我修正**的过程
### 损失函数与代价函数

在机器学习中必须量化错误有多大
- 损失函数 $L(\hat{y}, y)$用于衡量**单个**训练样本的预测准确度
- 代价函数$J(w, b)$：计算的是**全量**训练样本误差的平均值
#### 回归常用：均方误差（MSE）
预测的是一个**连续数字**（如房价、气温）时，通常使用 MSE
平方会放大较大的差距，这迫使模型尽可能减小那些巨大的偏差
- 单个样本： $$L(\hat{y}, y) = (\hat{y} - y)^2$$
- 所有样本：

$$J(w, b) = \frac{1}{2m} \sum_{i=1}^{m} (\hat{y}^{(i)} - y^{(i)})^2$$
#### 分类常用：交叉熵损失
预测的是**概率或类别**时，必须使用交叉熵
公式：
$$L(\hat{y}, y) = -[y \log(\hat{y}) + (1-y) \log(1-\hat{y})]$$
### 链式法则
计算函数对每一个参数 $w$ 和 $b$ 的梯度，使得总误差 $J$ 减小，微调 $w^{[1]}$
- **计算方向**：从输出层倒退回输入层。
    
    1. 计算输出层误差。
        
    2. 将误差分摊给上一层的神经元。
        
    3. 根据分摊到的误差，计算该层权重的更新量。

##  数据集的划分

在实际工程中，绝不能在同一份数据上既考试又练习

| **数据集** | **意义**                            |
| ------- | --------------------------------- |
| **训练集** | 算法通过这些数据调整自己的参数（权重）。              |
| **验证集** | 用来调优超参数（比如决定神经网络该用几层），防止模型“死记硬背”。 |
| **测试集** | 评估模型在从未见过的数据上的真实表现。**绝对不能参与训练**   |
